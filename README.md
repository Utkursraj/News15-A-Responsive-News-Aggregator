# News15-A-Responsive-News-Aggregator
News15 is a responsive web application that fetches and displays the latest news headlines using the NewsAPI. The app allows users to search for news articles by keywords, and displays them in an organized grid layout. The design is fully responsive, ensuring a great user experience across devices from desktops to mobile phones.

Features:
Live News Feed: Fetches top headlines from reliable sources using NewsAPI.
Search Functionality: Allows users to search for specific news topics.
Responsive Design: Optimized for various screen sizes with smooth layout adjustments.
Interactive Blog Cards: Clickable news cards that open the full article in a new tab.
Real-time Updates: Dynamically loads and displays news as it's fetched from the API.
Error Handling: Displays appropriate messages when no results are found or an error occurs during the API call.
Technologies Used:
HTML5, CSS3, JavaScript (ES6+)
NewsAPI for fetching news data
Flexbox and CSS Grid for layout
Responsive design techniques for mobile-first development
